Introduction

KOB stands for "key on base" (or "key on board").  Typically, a KOB includes a key and a sounder and is used by students for practicing Morse code — to get used to the feel of the key and the sound of the sounder.  A KOB can also be pressed into service as a portable instrument, since it's not screwed down permanently to a desk.

I developed the Morse KOB program for much the same purpose as the traditional KOB.  It served me well as a Morse practice set while I was learning the code, and I use it regularly to communicate with other operators over the Internet.

Morse KOB works with Windows 95 or later, including Windows XP.  The program includes the following features:

  • Simulates a sounder using the computer's speakers
  • Can drive a sounder via the serial port (requires a separate interface circuit)
  • Simulates a straight key or bug using the Ctrl and Shift keys on the keyboard
  • Accepts a straight key or bug connected directly to the serial port
  • Has a built-in keyer (automatic dots and manual dashes) for use with a paddle
  • Sends American Morse from text typed on the keyboard or copied from a file
  • Reads American Morse and displays the text
  • Compatible with CWCom and dial-up Morse

Getting started

Once you've installed the program (see Appendix A) try the following:

  • Press the Esc key to toggle the buttons in the Circuit Closer to the Open position.  This opens the KOB's circuit, and you should hear the clack of the simulated sounder deenergizing.
  • Press and release the Shift key.  You should hear a click and a clack from the speakers.
  • Click on the Options tab, then pull down the Key menu and select Keyer.  Press the Ctrl key.  You should hear a series of dots coming out of the speakers.
  • Click in the lower pane of the Morse KOB window to place the cursor there.  This is the text box for the code sender.  Type some text and press the space bar or Enter key.  You should hear American Morse being sent, and you should see the decoded text appearing in the upper pane (the text box for the code reader).

Code practice

If you're interested in learning American Morse, I highly recommend The Mill program, available at http://home.comcast.net/~w4fok/.  For additional code practice, however, Morse KOB can also be quite useful.

Open up a file on your computer (or an email message, or a web page), select the text you're interested in, copy it to the Windows clipboard, and then paste it into the KOB's code sender window by pressing the Insert key.  Press the Enter key when you're ready for the program to start sending Morse.

The Code Speed setting determines the speed in WPM of (a) the keyer, (b) the code sender, and (c) the approximate speed expected by the code reader.  You can reduce the code speed by pressing function key F4, and increase it by pressing F5.  (See Appendix B for a complete list of the program's keyboard commands.)

Hooking up a key and sounder

Although you can use your computer's Shift key to simulate a straight key – and even use the Ctrl and Shift keys to simulate the paddle of a bug – at some point you'll probably want to start using a real key instead.  You can do this by hooking up a straight key, bug, or keyer paddle to your computer's serial port.  See Appendix C for details on which pins to use.

You'll also need to tell the program which COM port you're using.  To do this, click on the Options tab, then pull down the Serial menu and select your COM port.  If you're not sure which one it is, try COM 1 or COM 2.

A sounder can also be connected to the serial port, but this requires a bit more work.  You'll need an interface circuit such as the one described at www.mrx.com.au/images/Sounder%20CCT.jpg.  The circuit can be built from parts available at Radio Shack for about $5, and the components will fit inside a DB9 serial port connector.


Serial port connector with sounder interface circuit inside

While we're on the subject of sounders, you can change the sound of the simulated sounder.  Go to the Options tab, pull down the Sounder menu, and select Snappy or Mellow.  Snappy has a crisp sound to it, while Mellow sounds like the sounder's in a resonator – maybe with a Prince Albert can behind it.  If you're used to CW, then the Tone option will sound more familiar.  The Mixed option provides a combination of tone and sounder, to help you make the transition to copying Morse by sounder alone.

Code reader

Once you've got a key hooked up to your computer, you can see how well the code reader copies your fist when you're sending American Morse.  The code reader gives a rough indication of the quality of your sending, but it sometimes misinterprets even well sent code so don't get discouraged if you see it making lots of errors.  It does do an excellent job of telling you if you've sent too many or too few dots, however.

The code reader is kind of like training wheels: useful while you're first learning, but unnecessary once you get the hang of it.  I still find it nice to have while I'm on the wire, though, as a "second opinion" when I'm not sure if I copied something right.

When you're using the code reader, the program's code speed should be set to roughly the same speed as the Morse that it's trying to decode.  Unrecognized code is displayed within brackets (for example [.-...] ).  The circuit closure between transmissions is represented by an underscore.

Internet Morse

Practicing Morse code by yourself can get boring after a while.  It's a lot more fun to use Morse as it was originally intended: as a way of communicating with other people.  You can do this either over the Internet or by using "dial-up Morse".

If your computer is connected to the Internet, simply go to the Internet tab and click on the Connect button.  Within a fraction of a second, the Connect indicator above the Connect button should go from white to red, showing that you've successfully connected to the Internet Morse server.  The server acts as a "hub", tying any number of offices together into a common telegraph "circuit".  Make sure your key is closed or the KOB's Circuit Closer is in the Closed position; otherwise you'll "break" the circuit as soon as you connect.

If the Connect indicator flashes red and white instead of showing a steady red, that means the server isn't responding to your connect request for some reason.

Once you've successfully connected to the server, you can listen to other operators on the wire or you can open your key and make a transmission, just as if you were on a real telegraph wire.  Remember to close your key at the end of each transmission, or press the Esc key to toggle the Circuit Closer to the Closed position.

Note: If some other operator on the wire leaves his key open (possibly because he's using CWCom instead of Morse KOB), when you open your key and try to send, your sounder will remain silent.  To regain control of the wire, press the Esc key twice to cycle the Circuit Closer to Closed then Open.

Within 30 seconds after connecting, in the text box to the left of the Connect button you'll see a list of the user IDs for everyone connected to the server.  Whenever someone transmits, his ID is moved to the bottom of the list.  So the station at the top of the list is the one that has waited the longest since his last transmission.  A station's ID is not automatically removed from the list after disconnecting, but you can edit the list manually using your mouse and keyboard.

When you're ready to end your session on the Internet "wire", simply click on the Disconnect button.

For more Internet options, please see Appendix E.

Dial-up Morse

Morse KOB allows you to use your computer's built-in modem to do dial-up Morse.  (See the Morse Telegraph Club's website for information on dial-up Morse.)

In order to use the dial-up Morse feature, your computer must have a V.253-compliant, full-duplex voice modem (such as the Modem Blaster PCI modem, available for $40 at Creative.com).  Furthermore, it uses the voice modem in a way that's on the fringe of what it was designed to do.  Although it works well on my computer (a Dell Dimension 4100 running Windows XP with a Creative Modem Blaster modem), there's no guarantee it'll work with yours.

If you'd like to try the dial-up modem feature, be sure to read Appendix F carefully, and contact me for additional help in getting started.

Appendix A: Installing the program

Morse KOB can be downloaded from my Morse page at www.morsekob.org.  The basic steps are as follows:

  1. Create a temporary folder on your computer.
  2. Save the download file in the temporary folder.
  3. Expand the compressed download file into the same temporary folder.
  4. Run setup.exe and follow the on-screen instructions.
  5. Once the program has been installed, you can delete the temporary folder.

Note: The installation process resets all of the program's options to their default values.  So if you have previously changed any of the settings in the Options tab, you should make a note of these settings so you can restore them after the new version is installed.

To run the program, go to the Morse KOB program folder from the Start menu and select Morse KOB.  The Morse KOB window should appear on the screen.

You can permanently remove the Morse KOB program and its associated data files from your computer by going to the Windows Control Panel and running the Add or Remove Programs utility.

The following table identifies the RS232 lines used by the program to interface with an external key or sounder.  There doesn't seem to be any standard color code for the wires in a serial cable, but the table includes two schemes that you might encounter.

Warning: Connecting a key to a serial port can expose your computer to possible damage if you touch the key when your body is charged with static electricity.  Although unlikely, it has happened, so take appropriate precautions.  For example, using a USB-to-serial adapter cable instead of connecting directly to the computer's serial port can limit the extent of possible damage to something that's easily replaced.

*The functions of the DTR and RTS lines are interchanged when the External Sounder option is set to Alternate (see Appendix D).

If your computer has a 9-pin serial port connector, then you can connect a straight key or bug to pins 4 and 6.  Be sure to set the Key option on the Options tab to Manual.

A keyer paddle can be connected to pins 4, 6, and 8.  Set the Key option on the Options tab to Keyer.

The dot paddle (normally the left paddle) sends dots automatically at whatever speed you specify in the Code Speed setting, while the dash paddle is used for sending dashes manually.  This allows you to use a keyer paddle (either "iambic" or single lever) to simulate a traditional bug, giving you a way to send longer dashes for L and Ø.

Appendix E: Internet Morse

Morse KOB allows you to connect with other Morse operators via the Internet, using the same method as MRX Software's CWCom system.  If you simply click on the Connect button on the Internet tab, you'll be connected to the default MTC server at mtc-kob.dyndns.org, using channel 1005 (the channel used for the regular MTC net), with a default ID of "KOB".  The server acts like a "hub", tying many users together on a telegraph "circuit".

The first thing you'll probably want to do is click on the Server tab and change the ID field to match your own office call.

The server allows you to tie into any of more than 60,000 independent circuits, known as Channels.  Using the Server tab, you can switch to a different channel if you wish.

You can also choose a different server.  However, please do not use the CWCom server at morsecode.dyndns.org at this time.  Morse KOB has not been certified for use on this server yet.

Appendix F: Dial-up Morse

The controls on the Dial-Up tab set the modem to Originate or Answer mode and Connect the modem to the phone line.

A signal indicator provides a graphical display of the last half second of the received signal when the modem is connected, giving an indication of the quality of the connection.

The connect indicator above the Connect button shows the status of the connection using the following color code:

White Disconnected
Black Connecting
Yellow Synchronizing
Red Connected

To establish a dial-up connection, dial the number using your telephone and click on the Connect button.  Within five to ten seconds after clicking the Connect button, the connect indicator should turn red, indicating that the modem is properly connected to the phone line.  At this point, you can hang up the telephone.

To disconnect, click the Connect button again.
