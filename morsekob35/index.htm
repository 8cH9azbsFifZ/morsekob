MorseKOB 3.5

The development of MorseKOB 3.5 is currently on hold, pending completion of other projects

System requirements

  • Windows 7, Vista, or XP
  • Microsoft .NET Framework 4: download from www.microsoft.com/download/en/details.aspx?id=17851
  • Microsoft Visual C++ 2010 SP1 Redistributable Package (x86): download from www.microsoft.com/download/en/details.aspx?id=8328
  • (XP only) Microsoft DirectX End-User Runtimes: download from http://www.microsoft.com/en-us/download/details.aspx?id=8109

Hopefully some day I'll be able to come up with a streamlined installation process that loads these prerequisites automatically, but for now you have to do it yourself. Sorry about that...

Installation

  1. Send me an email at the address at the bottom of the MorseKOB home page, requesting the latest distribution file. (The beta version isn't available yet, but if you like I'll add your name to the waiting list and let you know when it's ready.)
  2. Open the distribution file.
  3. Copy the contained MorseKOB 3.5 folder to your hard drive.
  4. Run the program by double-clicking on the MorseKOB.exe application file.  (It's the one with the icon.)

If you wish, you can create a desktop shortcut by right-clicking on the MorseKOB.exe file and selecting Send to > Desktop.
